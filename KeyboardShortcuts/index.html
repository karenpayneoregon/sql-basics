Visual studio common keyboard shortcuts

Comment selection
ctrl + K + C
Comment the selected code.
Un-Comment selection
ctrl + K + U
Uncomment the selected code.
Locate current file in solution explorer
ctrl + alt + L
Opens the solution explorer and highlights the current file.
Duplicate current line
ctrl + D
Duplicates the current line or selection.
Expand selection
shift + alt + =
Incrementally expands the selection based on the syntax of the code.
Surround with
ctrl + S
Surrounds the selected code with a snippet e.g. for, try/catch
Move line up
alt +
for a line or selection, moves it up.
Move line down
alt +
for a line or selection, moves it down.
Toggle breakpoint
F9
Toggles a breakpoint on the current line of code.
Reopen last closed document
ctrl + K + Z
Reopens the last closed document in the editor.
Feature specific help
ctrl + shift + P
Searches for the focused feature and activate if found.
